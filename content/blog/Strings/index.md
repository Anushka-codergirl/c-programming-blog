## String Manipulation Function

| S.L.No | Function                            |                             Use |
| :----- | :---------------------------------- | ------------------------------: |
| 1      | strlen(str)                         |         omputes string's length |
| 2      | strcpy(str destination ,str source) |      copies a string to another |
| 3      | strcat(str1,str2)                   | concatenates(joins) two strings |
